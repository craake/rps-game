# Rock-Paper-Scissors game

## Requirements
- Ruby 2.3+
- Ruby on Rails 5.2.1

## Running tests
Run in your terminal 
```
bundle exec rspec
```
## Installation
- Run in your terminal
```
bundle install

RAILS_ENV=prod rake assets:precompile

rails s -e production
```
- Open your browser, and navigate to [**http://localhost:3000**](http://localhost:3000)
