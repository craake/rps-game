# Rock-Paper-Scissors game

## Requirements
- Ruby 2.3+
- Ruby on Rails 5.2.1

## Installation
Run the following in your project dir
```
bundle install
```

## Running tests
Run the following in your project dir 
```
bundle exec rspec
```
## Usage
- Run the following in your project dir
```
rails s
```
- Open your browser, and navigate to [**http://localhost:3000**](http://localhost:3000)
